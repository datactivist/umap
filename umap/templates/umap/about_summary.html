Je veux accéder à une carte existante.
Je cherche cette carte dans la barre de recherche

Je veux créer une carte, je clique sur le bouton “Créer une carte”.
J'ai la possibilité de :

  • Importer des données proposées par défaut dans l'outil
  • Importer des données locales
  • Importer des données via une URL
  • Choisir un fond de carte
  • Choisir la couleur et les icônes
  • Gérer les couches de données
  • Choisir la licence de vos données importées
  • Exporter et partager votre carte

Et c'est un logiciel libre !
